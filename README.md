## 简介

复刻了微信的多种界面与功能，包括：

- 通讯录、对话（私聊 & 群聊）
- 发送各种类型的消息：文本、图片、转账、红包、消息引用、语音
- 发布 / 编辑朋友圈
- 钱包、交易记录
- …更多功能持续开发中

支持进入**编辑模式**进行所见即所得的修改，浏览器端可直接截图。

## 快速开始

### 前置要求

- Node.js >= 18
- pnpm >= 9

### 安装 & 运行

```bash
# 克隆仓库
git clone https://github.com/caterpi11ar/wechat-online.git
cd wechat-online

# 安装依赖
pnpm install

# 启动开发服务器
pnpm run dev
```

### 构建

```bash
pnpm run build
```
